# Portfólio GitHub

Site de portfólio pessoal pronto para publicar no GitHub Pages.

## Como usar

1. Edite `script.js` e ajuste `githubUsername` para seu usuário.
2. Edite `script.js` no bloco `customProjectBriefs` para personalizar a proposta e o objetivo de cada repositorio.
3. Edite `index.html` para atualizar seu nome, bio e contatos.
4. Faça commit e push para a branch principal.
5. Ative o GitHub Pages em `Settings > Pages`.

## Estrutura

- `index.html`: estrutura da página.
- `styles.css`: estilo visual e responsividade.
- `script.js`: busca, enriquece e renderiza projetos do GitHub.
