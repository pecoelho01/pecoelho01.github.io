# Portfólio GitHub

Site de portfólio pessoal pronto para publicar no GitHub Pages.

## Como usar

1. Edite `script.js` e ajuste `githubUsername` para seu usuário.
2. Edite `index.html` para atualizar seu nome, bio e contatos.
3. Faça commit e push para a branch principal.
4. Ative o GitHub Pages em `Settings > Pages`.

## Estrutura

- `index.html`: estrutura da página.
- `styles.css`: estilo visual e responsividade.
- `script.js`: busca e renderiza projetos do GitHub.
